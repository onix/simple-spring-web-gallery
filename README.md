# Simple Spring Web gallery #

Simple Spring MVC Web gallery with user registration, authentication, photo upload and photo viewing.

Built with Spring Framework, Spring Security, PostgreSQL 9.1 and Twitter Bootstrap. Adopted for Heroku cloud. Front-end coded with Twitter Bootstrap.

This software includes 3rd party [image utility library](http://www.gotoquiz.com/web-coding/programming/java-programming/resize-images-in-java-preserving-image-quality/ "Resize images in Java, preserving image quality") by gotoquiz.com.
